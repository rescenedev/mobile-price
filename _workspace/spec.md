# Spec — Phase 0 Pre-flight Survey

## Project
- **Slug**: `ratsaver` (요금제 세이버)
- **One-liner**: 휴대폰 요금제(알뜰폰·통신사)를 조건으로 검색·비교하고 절약액을 계산해 최적 요금제를 추천하는 공개 비교 웹.
- **Date**: 2026-06-14

## 콘셉트
국내 휴대폰 요금제를 한 곳에서 조건 필터로 검색하고, 상세/비교/맞춤추천/절약계산까지 제공하는 가볍고 빠른 공개 비교 사이트. 로그인 없이 누구나 사용.

## 타깃
- 알뜰폰/요금제 갈아타기를 고민하는 일반 소비자
- "내 사용량 기준 가장 싼 요금제"를 빠르게 알고 싶은 사람

## 데이터 모델
- **소스**: D1 시드 데이터(큐레이션). 외부 API 의존 없음.
- **규모**: 요금제 50~150개
- **핵심 엔티티**: `plan`
  - 통신사(carrier), 망(network: SKT/KT/LGU+), 요금제명(name)
  - 월요금(monthlyPrice, 원), 데이터(dataGb / 무제한 여부), 통화(callMinutes / 무제한), 문자(smsCount / 무제한)
  - 데이터 소진 후 속도(throttleKbps), 약정 여부(contract), 통신사 유형(mvno: 알뜰폰 여부)
  - 프로모션/비고(notes)

## 인증
- **없음** (공개 비교 사이트). 로그인 게이트 없음. 즐겨찾기는 추후 localStorage 검토(MVP 제외).

## 수익화
- MVP 범위 아님. 추후 제휴/CTA(가입 링크) 자리만 비워둠.

## 핵심 기능 (4)
1. **조건 필터 검색** — 망/통신사/데이터/통화/문자/가격대/알뜰폰여부 필터 + 정렬(가격·데이터)
2. **요금제 상세 + 비교** — 상세 페이지 + 2~3개 나란히 비교 테이블
3. **맞춤 추천** — 월 사용량(데이터/통화) 입력 → 점수 매칭 → 추천 카드
4. **절약액 계산기** — 현재 월요금 입력 → 추천 요금제 대비 절약 가능액

## 렌더링 기본 성향
- 목록/홈/상세: **정적 우선** (시드 데이터 → SSG/ISR). 추천·계산기·필터는 클라이언트 인터랙션 또는 edge.
- perf 민감도: **높음** (비교 사이트는 속도가 곧 이탈률).

## perf 예산 (북극성, Phase 2에서 수치 확정)
- API p95 < 200ms (기본)
- LCP < 1.5s / INP < 200ms / CLS < 0.1
- 라우트별 JS 번들 < 200KB gz

## Cloudflare
- 계정/도메인: 배포 단계(Phase 7)에서 확인. MVP는 로컬 preview까지 보장.
- 바인딩: D1(요금제) · KV(CACHE) · AE(관측). R2 불필요.

## 비범위 (Out of scope, MVP)
- 실시간 외부 API, 사용자 계정, 결제/제휴 정산, 관리자 CMS(시드는 SQL/스크립트로 관리)
